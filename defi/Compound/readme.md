# Compound

是一个以太坊上的货币市场，一个任何用户、机构和 dApps 都可以使用的链上账本。它提供了存币和借币的功能，就像一个银行，用户可以存币获的利息收益，或进行抵押借币。在实现原理上，Compound 的帐本模型也与银行类似，并遵循了国际会计准则。
基础特性：
● 资金池
● 基于供需法则，由算法生成利率
● 浮动利率，无需协商
● 完全透明的代币余额信息，记录所有历史利率

利率模型：
U = 借款 /（ 现金 + 借款 ）
借款利率 = 10% + U *30%
存款利率 = 借款利率* U *（ 1 - S ）
